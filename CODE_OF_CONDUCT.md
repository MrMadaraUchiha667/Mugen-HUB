# Code of Conduct

## Our Pledge

We are committed to creating a welcoming and inclusive environment for everyone.

---

## Expected Behavior

- Be respectful.
- Help newcomers.
- Accept constructive criticism.
- Collaborate professionally.

---

## Unacceptable Behavior

- Harassment
- Hate speech
- Personal attacks
- Spam
- Discrimination

---

## Enforcement

Project maintainers may remove comments, issues or pull requests that violate this Code of Conduct.
